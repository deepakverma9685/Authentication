üîó API Endpoints

POST /api/v1/auth/register

Description: Register a new user account

Request Body:

{ "firstName": "John", "lastName": "Doe", "email": "john.doe@example.com", "password": "SecurePassword123!" }

Response: User object with ID and email verification status

POST /api/v1/auth/login

Description: Authenticate user and receive JWT tokens

Request Body:

{ "email": "john.doe@example.com", "password": "SecurePassword123!" }

Response: JWT access token, refresh token, and expiration time

POST /api/v1/auth/refresh

Description: Refresh JWT access token using refresh token

Request Body:

{ "refreshToken": "your_refresh_token_here" }

Response: New access token and refresh token

POST /api/v1/auth/forgot-password

Description: Initiate password reset process

Request Body:

{ "email": "john.doe@example.com" }

Response: Success message (email sent if account exists)

POST /api/v1/auth/reset-password

Description: Complete password reset using token from email

Request Body:

{ "token": "reset_token_from_email", "newPassword": "NewSecurePassword123!" }

Response: Success message
